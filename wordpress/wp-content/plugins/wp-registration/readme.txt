WordPress Simple Registration Form Plugin

== Description ==
A Simple WordPress plugin to turn your site into a membership site. It will generate all necessary pages required for
any membership site. Awesome drag and drop field editor creates signup page so quickly. Members can manage profile on 
frontend. For More Detail See Video Below:

[vimeo https://vimeo.com/289290541]
[See More Videos](https://najeebmedia.com/videos-tutorial-for-wordpress-simple-registrations/)

Since version 4.3, Admin can message via email to selected members from Dashboard with subject. See screenshot-16


== Installation ==

1. Upload plugin directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. After activation, you can set options from `WP Register` menu
